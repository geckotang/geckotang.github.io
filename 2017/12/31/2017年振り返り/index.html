2017年振り返り

2017年の振り返り。せっかくブログ作ったし書いておく。

技術とか

  • CSS Grid元年にふさわしく、CSS Gridに関する記事をよく目にするようになった。
    • IEが死ぬまで便利には使えないのでそこをどうしていくかが難しい
    • 来年は案件で使ってみたい
  • MacVimからVSCodeに変えたものの結果的にVimR（neovim）へ。やはりVimである。
    • dein.vimのおかげでプラグインの管理がより楽になった
  • MacBook Pro (15-inch, Late 2016)のキーボードが2回壊れた
    • よくあるやつ。へこみっぱなしで返ってこないとか、反応が悪いとか。
    • 初回は保証期間内だったから無料だった。2回めは5万円の修理代。
    • 同じような症状の人々が多く、これはリコールなのではと思いつつ、まぁいいや。
  • 左右分離式キーボードを買った。
    • Barocco
    • MBPのキーボードが壊れていた時期に買った。
    • 左右分離してるおかげで腕が広がって良い。家では使っていく。
    • 矢印キーがないからちょっと慣れない。

仕事とか

  • 案件でCSS Gridを使うことがなかった。IEめ。
  • CodeGridの記事をたくさん書いた。14本くらい。
    • 来年もたくさん書いてこ。
  • 今やってる案件では、ECSSで頑張ってる。
    • コードを捨てられることを意識してmixinやextendは使わない。
    • 依存を極力減らしていくようにしている。
    • どう考えてもコードの重複は多いが安心して捨てられる。
    • コンポーネントの粒度はJS側に合わせている。

登壇と勉強会とか

  • builderscon tokyo 2017でCSS Gridの話をしました。
  • 来年はもう少し登壇しようかな…
  • W3C CSS Module 仕様書もくもく会@東京はほぼ毎月出てた。もう26回も開催している。続けていきたい。
